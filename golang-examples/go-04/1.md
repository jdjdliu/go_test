## 1.4.1 与其他主要编程语言的差异

&^按位置清零

1 &^ 0 -- 1
1 &^ 1 -- 0
1 &^ 1 -- 0
0 &^ 0 -- 0

结论：右边是1，结果是0。右边是0，左边是什么就是什么。
